**Variáveis de Ambiente**

Para que a integração da aplicação com o banco de dados e a api de pagamento funcione corretamente devemos configurar algumas variáveis de ambiente, após inicializar os conteiners use os comandos:

docker exec -it <id_do_conteiner_web> bash #Para acessar o terminal do conteiner 

#Adicione um arquivo .env na pasta raíz do projeto com as variáveis de ambientes de acesso ao postgres

vim .env

# Dentro do arquivo adicione as seguintes variáveis:
POSTGRES_USER=<user> \
POSTGRES_PASSWORD=<password>\
POSTGRES_DB=<db> \
DATABASE_URL=postgres://postgres:<password>@db:5432/<db> \
SECRET_KEY_BASE=$(rails secret) \


# Para adicionar credenciais encriptadas de acesso a api do MP, use os comandos:

EDITOR=vim bundle exec rails credentials:edit #Para acessar as credenciais/variáveis encriptadas pelo rails

# Dentro do arquivo adicione:

mercadopago:
    token: "<token_de_produção_da _sua integração_MP>"
    user_id: "<user_integração_mp>"
    external_pos_id: "<id_do_caixa_de_integração_MP>"
    
caso tenha dúvidas referentes a estes tokens, acesse: https://www.mercadopago.com.br/developers/pt/docs
